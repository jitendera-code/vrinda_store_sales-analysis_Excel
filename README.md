# 📊 E-commerce_ Sales Intelligence _Dashboard (Excel)

## 🔍 Overview

This project focuses on analyzing retail sales data to generate actionable business insights. The dashboard helps identify key revenue drivers, customer segments, and regional performance trends.

## 🎯 Business Objectives

* Identify top-performing customer segments
* Analyze regional sales distribution
* Track monthly revenue trends
* Evaluate product category performance

## 🛠️ Tools & Techniques

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning & Transformation
* Interactive Dashboard Design

## 📊 Key Insights

* Female customers contribute the majority of revenue
* A few states dominate overall sales performance
* Seasonal trends significantly impact monthly revenue
* Certain product categories generate higher profitability

## 📈 Dashboard Features

* KPI Metrics (Total Sales, Orders, AOV)
* Monthly Sales Trend Analysis
* Regional Sales Distribution
* Customer Segmentation Insights
* Category-wise Performance
